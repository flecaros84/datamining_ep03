# EP03 - Proyecto de Data Mining

Proyecto de minería de datos orientado al análisis de abandono de clientes residenciales en una empresa de telecomunicaciones.

## Contenido del repositorio

- `EP03.ipynb`: notebook principal del proyecto.
- `docs/`: pauta del trabajo y descripción del dataset.
- `data/`: datos originales utilizados en el análisis.
- `data/processed/`: archivos generados durante la preparación y segmentación de datos.

## Objetivo

Identificar patrones asociados al abandono de clientes residenciales mediante análisis exploratorio, preparación de datos, modelos supervisados de clasificación y segmentación no supervisada.

## Enfoque aplicado

El trabajo considera:

- comprensión del negocio;
- comprensión y exploración de datos;
- preparación del dataset;
- modelamiento supervisado con Regresión Logística, SVM lineal y Árbol de Decisión;
- segmentación no supervisada mediante K-Means;
- conclusiones y recomendaciones de negocio.
